2. Zweck der Website

Diese Website dient der Bereitstellung einer Community-Lootliste für Diablo II: Resurrected. Nutzer können Items einsehen und ihr Interesse bzw. ihren Bedarf an Items bekunden.

3. Verarbeitete Daten

Im Rahmen der Nutzung können folgende personenbezogene Daten verarbeitet werden:

  • frei gewählter Nickname
  • Kontaktangaben (z. B. E-Mail-Adresse oder vergleichbare Angaben)
  • Inhalte von Bedarfs- oder Item-Anfragen
  • technische Nutzungsdaten (z. B. Zeitpunkte von Aktionen)

Die Verarbeitung dieser Daten erfolgt ausschließlich zur Organisation und Abwicklung von Item-Anfragen innerhalb der Community.

4. Speicherung und technische Verarbeitung

  • Bestimmte Informationen (z. B. Status von Items, Bedarfsangaben, Besucherzähler) werden lokal im Browser mittels localStorage oder vergleichbarer Webspeicher-Techniken gespeichert.
  • Zur technischen Bereitstellung der Lootliste können Item-Daten und Bedarfsinformationen über ein externes Backend verarbeitet werden.

Eine Weitergabe der Daten an Dritte zu kommerziellen Zwecken erfolgt nicht.
