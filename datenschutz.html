2. Zweck der Website

Diese Website dient der Bereitstellung einer Community-Lootliste für Diablo II: Resurrected. Nutzer können Items einsehen und über ein Anfrageformular Interesse an Items bekunden.

3. Verarbeitete Daten

Im Rahmen der Nutzung können folgende personenbezogene Daten verarbeitet werden:

  • frei gewählter Nickname
  • Kontaktangaben (z. B. E-Mail-Adresse oder Discord-Name)
  • Inhalte von Item-Anfragen

Diese Daten werden ausschließlich zur Organisation und Abwicklung von Item-Anfragen innerhalb der Community verwendet.

4. Speicherung und technische Verarbeitung

  • Bestimmte Informationen (z. B. Anfragen, Besucherzähler) werden lokal im Browser mittels localStorage gespeichert.
  • Serverseitig erfolgt eine Verarbeitung von Item-Daten und Anfragen über ein externes Backend (Hosting bei Railway).

Eine Weitergabe der Daten an Dritte zu kommerziellen Zwecken erfolgt nicht.

5. Hosting

  • Frontend: GitHub Pages
  • Backend/API: Railway

Die Anbieter verarbeiten Daten ausschließlich im Rahmen ihrer Auftragsverarbeitung gemäß Art. 28 DSGVO.

6. Cookies & Tracking

Diese Website verwendet keine Cookies zu Marketing- oder Tracking-Zwecken. Es kommen keine externen Analyse- oder Werbedienste zum Einsatz.

7. Rechte der betroffenen Personen

Nutzer haben gemäß DSGVO das Recht auf:

  • Auskunft über gespeicherte Daten
  • Berichtigung unrichtiger Daten
  • Löschung personenbezogener Daten
  • Einschränkung der Verarbeitung

Anfragen hierzu können jederzeit per E-Mail an den Verantwortlichen gerichtet werden.

8. Freiwilligkeit der Angaben

Die Angabe personenbezogener Daten erfolgt freiwillig. Ohne Angabe bestimmter Daten (z. B. Kontaktmöglichkeit) ist eine Item-Anfrage jedoch nicht möglich.
